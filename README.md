## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Routes
<strong>GET</strong>
<ul>
  <li><strong>/post/:id</strong>: Fetch a single post by its id</li>
  <li><strong>/feed</strong>: Fetch all published posts</li>
  <li><strong>/filtered-posts/:searchString</strong>: Filter posts by title or content</li>
</ul>
<strong>POST</strong>
<ul>
  <li><strong>/post</strong>: Create a new post
    <ul>
      <li>Body:
        <ul>
          <li>title: String (required): The title of the post</li>
          <li>content: String (optional): The content of the post</li>
          <li>authorEmail: String (required): The email of the user that creates the post</li>
        </ul>
      </li>
    </ul>
  </li>
  <li>/user: Create a new user
    <ul>
      <li>Body:
        <ul>
          <li>email: String (required): The email address of the user</li>
          <li>name: String (optional): The name of the user</li>
        </ul>
      </li>
    </ul>
  </li>
</ul>
<strong>PUT</strong>
<ul>
  <li><strong>/publish/:id</strong>: Publish a post by its id</li>
</ul>
<strong>DELETE</strong>
<ul>
  <li><strong>/post/:id</strong>: Delete a post by its id</li>
</ul>
